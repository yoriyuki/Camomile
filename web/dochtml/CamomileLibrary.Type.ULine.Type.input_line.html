Class CamomileLibrary.Type.ULine.Type.input_line


class input_line : CamomileLibrary.Type.UChar.t
       #CamomileLibrary.Type.OOChannel.obj_input_channel -> [text] CamomileLibrary.Type.OOChannel.obj_input_channel
new input_line input_obj creates the new input channel object CamomileLibrary.Type.OOChannel.obj_input_channel which reads Unicode characters from input_obj and output lines. All of CR, LF, CRLF, NEL, LS, PS, as well as FF (formfeed) are recognised as a line separator.
