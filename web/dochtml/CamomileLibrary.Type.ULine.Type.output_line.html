Class CamomileLibrary.Type.ULine.Type.output_line


class output_line : ?sp:CamomileLibrary.Type.ULine.separator -> CamomileLibrary.Type.UChar.t
       #CamomileLibrary.Type.OOChannel.obj_output_channel -> [text] CamomileLibrary.Type.OOChannel.obj_output_channel
new output_line ~sp output_obj create the new output channel object CamomileLibrary.Type.OOChannel.obj_output_channel which output each line to output_obj using sp as a line separator. If sp is omitted, linefeed (LF) is used.
